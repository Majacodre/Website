About
This website gathers data from Atlas of Surveillance to spread and raise awareness about surveillance technologies that are used across the US. The data is presented in a graph and a map to illustrate the number of different surveillance tools used per state.
What is Atlas of Surveillance
Atlas of Surveillance is a collaboration between the Electronic Frontier Foundation and the University of Nevada, Reno Reynolds School of Journalism. It consists of a map that provides information about different law enforcement agencies’ surveillance technologies. The website's goal is to create an open source that provides data about what’s been purchased locally and how technologies are spreading across the country.
