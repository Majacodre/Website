Purpose:
This website gathers data from Atlas of Surveillance, the data represents different technologies used by law 
enforcement agencies across different USA states. Purpose of this website is to present data visually from 
Atlas of Surveillance in order to show differences between technologies used in different states of USA and raise 
awareness about use of such tools. Map was created to show the most used surveillance technology by 
enforcement agencies in each state. The website has a simple look to provide a satysfing user experience in terms
of navigation.
 
Sources:
